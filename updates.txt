Hand Gesture Recognition & Communication Support System
1. Project Overview
SignVerax is a complete end-to-end machine learning system designed for real-time hand gesture recognition using computer vision and deep learning. The system detects sign language gestures, converts them into meaningful words, and builds communication sentences through an intuitive web interface.
The project follows industry-standard ML workflow from raw image collection → landmark extraction → model training → TFLite deployment → real-time web application, enabling non-verbal communication support for accessibility.

11. Core Technical Logic
1. Problem Framing
Multi-class gesture classification: 8+ hand gestures → communication words
2. Data Representation (Landmarks → Numbers)
text
Method: MediaPipe Hand Landmarks (21 points)
Raw: [x1,y1,x2,y2,...,x21,y21] ∈ [0,1]×42
Preprocessing:
1. Pixel: lm.x × width, lm.y × height
2. Relative: subtract wrist (lm0)
3. Normalize: divide by max(|vector|)
Output: 42D scale-invariant feature vector
3. Core Prediction Model
text
Architecture: MLP (42 → 128 → N_classes)
Activation: ReLU → Softmax
Loss: Categorical Crossentropy
Output: P(gesture_i | landmarks)
4. Real-time Inference Pipeline (60+ FPS)
text
Webcam → MediaPipe(21 landmarks) → Preprocess(42D) 
→ TFLite predict() → argmax() → Gesture label
5. Sentence Formation Logic
text
Rule-based grammar:
[doctor,warn,skin] → "The doctor warns about skin problems"
[you,afraid,pain] → "You are afraid because of pain"
Fallback: Natural language joining
